# Nintendo — Stardew Valley

Maquetación educativa en HTML, CSS y JavaScript vanilla basada en el proyecto Nintendo de Frontend Practice.

## Funciones

- Diseño responsive fiel a la referencia.
- Buscador desplegable.
- Galería con miniaturas y lightbox.
- Botones de favoritos animados.
- Carrusel responsive de juegos.

El proyecto anterior de Star Wars se conserva dentro de `star-wars/`.
